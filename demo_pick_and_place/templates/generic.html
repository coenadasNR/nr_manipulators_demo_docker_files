Pick and Place Demo

This project is for the National Robotarium (NR) Manipulator Setup manipulation tasks. The demos showcase the integration of robotic manipulation with digital twin technology, utilizing the following key components:

    • xArm5 Robotic Arm - A 5-axis manipulator that performs the physical pick-and-place operations once a cubeâ€™s identity has been verified.
    • Realsense RGB-D Camera - Captures ArUco code printed on every cube, enabling code decoding.
    • OpenCV Vision Pipeline - Extracts the ArUco marker code of each cube and decodes it, ensuring the payload match before handing the target pose to the planner.
    • MoveIt 2 - Generates collision-free trajectories for the xArm5 based on the poses produced by the vision pipeline.
    • Isaac Sim - NVIDIA's physics-based digital-twin environment allows us to model the camera, arm, and conveyor, enabling validation of the vision and manipulation stack under realistic lighting and dynamics.
    • ROS 2 - Provides the middleware that links vision, planning and arm control, allowing the exact same nodes to run in both simulation and on the physical cell.
